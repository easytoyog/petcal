# In the Park Release Notes

## What’s New In This Release
- Added a new `For You` experience with park search, liked parks, and dog-owner news
- Added client-side dog news and tips from sources like Reddit, AKC, PetMD, and Dogster
- Improved park search to better match partial park names
- Added stronger dog-park identification and dog-style map markers
- Improved map recenter reliability
- Added private pet document storage with expiry dates
- Added richer dog profile fields, including sex, size, weight, birthday, and temperament
- Added group chat notifications and unread indicators
- Removed email verification friction from signup and switched to confirm email / confirm password
- Improved the Parks tab layout and search experience
- Added a `Back to Top` action beside `Start a Walk`

## User-Facing Improvements
- Parks search is more forgiving for partial names
- News cards now open inside the app instead of sending users out to the browser
- Dog-owner news is fresher and cleaner in the feed
- Pet documents are now owner-viewable only
- Group updates and chats feel more responsive and easier to follow

## Quality and Performance Improvements
- Reduced several Firebase cost hotspots
- Tightened push notification and group message handling
- Improved park auto-checkout logic for stale check-ins
- Removed duplicate token persistence behavior
- Improved overall tab polish in Parks and profile areas

## Short Version
This update makes In the Park more useful day to day with better park search, a new dog-news experience, private pet documents, richer dog profiles, improved group chat notifications, and a smoother signup flow.
